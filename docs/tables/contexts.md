# Tables and Contexts

## Introduction

Ishi supports several CSS context classes:

* `attention`
* `info`
* `success`
* `warning`
* `danger`
* `light`
* `dark`

You can add an `alert` class too for added emphasis.

You can apply these context classes to `<table>`.

## Styling The Whole Table
